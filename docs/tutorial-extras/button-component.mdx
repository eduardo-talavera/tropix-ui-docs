# Button

## Button props
- `variant: ` → `string options: 'primary' | 'success' | 'warning' | 'danger | ghost'`
- `size: ` → `string options: 'sm' | 'md' | 'lg' | 'full'`
- `loader` → `boolean`
- `outline` → `boolean`
- `rounded` → `boolean`


````mdx-code-block
import { ButtonVariants } from '@site/src/components/button/ButtonVariants';
import { ButtonSizes } from '@site/src/components/button/ButtonSizes';
import { ButtonLoader } from '@site/src/components/button/ButtonLoader';


## Variant prop

```jsx
import { Button } from 'tropix-ui'


const ButtonVariants = () => {
  
  const buttonVariants = ['primary', 'success', 'warning', 'danger'];

  return <>
    {
      buttonVariants.map((variant) => (
          <Button 
            key={variant} 
            variant={variant}
            style={{ margin: '5px' }}
          >
            {variant}
          </Button>
        ))
    }
  </>
};
```
````
